    Structure

    The project follows Unity's recommend file and folder format, albeit with a few changes:

    <root>
  ├── package.json
  ├── README.md
  ├── CHANGELOG.md
  ├── LICENSE.md
  ├── THIRD PARTY NOTICES.md
  ├── .gitignore
  ├── Runtime
